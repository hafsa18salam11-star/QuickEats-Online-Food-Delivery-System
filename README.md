# QuickEats-Online-Food-Delivery-System
QuickEats is a database-driven web application that simplifies the food ordering experience by bridging the gap between customers and restaurants. Built with Python, Flask, and MongoDB, the system provides a seamless platform for browsing restaurants, exploring menus, managing orders, and processing payments, all in one place.

Tech Stack: MongoDB | Flask | Python | HTML | CSS | JavaScript

## Core Modules:

User Authentication & Profile Management.
Restaurant Browsing & Menu Search.
Cart Management & Order Placement.
Secure Payment Processing.
Order History & Status Tracking.
